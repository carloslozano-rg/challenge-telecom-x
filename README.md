## Descripción del Proyecto
El proyecto propone una situación ficticia donde hemos sido contratados como analistas de datos para trabajar específicamente en el 'churn' o cancelaciones de clientes de la empresa Telecom X.

La empresa está enfrentando un alto índice de evasión o cancelaciones de clientes y aún no han identificado el problema de esta evasión. 

Para ello, nos han proporcionado algunos datos y **nos han asignado el objetivo de extraer, tratar y realizar un análisis exploratorio de los datos**, y una vez limpios los datos, proporcionárselos al equipo de ciencia de datos. 

Adicionalmente **se nos han pedido los insights más relevantes, así como algunas recomendaciones**.


El **flujo de trabajo** se baso en el Trello proporcionado por Alura: <https://trello.com/b/hJk1ior5/telecomxlatam>, por lo que se puede llegar un resultado similar a la libreta **'TelecomX_LATAM.ipynb'** siguiendo dicho flujo.

## Comenzando

### Pre-Requisitos
- Puedes elegir cualquier editor de código fuente para trabajar con este repositorio.
Mi recomendación es usar **Visual Studio Code**.
- Python 3+.
- Git 2.5.0+ (Opcional, para llevar un mejor control de versiones).

### Instalación

Para obtener una copia de este reposirorio puedes:

1. Dirigirte a la url donde se encuentra alojado el [repositorio](https://github.com/carloslozano-rg/challenge-telecom-x.git) y decargarlo como archivo comprimido zip seleccionando el botón verde 'Code' y después 'Download ZIP'. ![Imagen de descarga de repo](assets/descarga-de-repo-imagen.png)

2. Dirigirte a tu carpeta local donde quieres alojar el repositorio y usa el siguiente comando en tu terminal: 

    ```
    git clone https://github.com/carloslozano-rg/challenge-telecom-x.git
    ```

Para instalar las librerías necesarias para trabajar con el repositorio, ejecuta este comando en tu terminal:

```
pip install -r requirements.txt
```

O si lo prefireres, puedes simplemente descargar la libreta **TelecomX_LATAM.ipynb**, y exportarla a un entorno en línea como [Google Colab](https://colab.research.google.com/) donde no requeriras descargar nada adicional.
